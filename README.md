# AryanMovie Discovery Website

## 📄 Overview
This is a **movie discovery platform** that allows users to search and browse movies and TV shows using **TMDB API**.  
The platform **does not host any movies or videos**; all content is streamed or linked from external legal sources. This project is built for educational, demo, and monetization purposes.

---

## ⚙️ Features

- 🔍 Search for movies and TV shows by title
- 🎬 Watch official trailers only
- ⭐ View ratings, cast, descriptions, genres
- ❤️ Save favorite movies
- 📂 Browse by categories: genre, year, rating
- 🌐 Fully responsive for desktop and mobile

---

## 🛠 Technology Stack

- **Frontend:** HTML, CSS, JavaScript (custom-built)
- **APIs:** [TMDB API](https://www.themoviedb.org/)
- **Hosting Options:** Netlify, Cloudflare Pages, Vercel
- **Optional Libraries:** (e.g., Axios, Swiper, etc., depending on implementation)

---

## ⚠️ Disclaimer

- This website **does not host copyrighted movies or videos**.
- All media content is retrieved from **external legal sources** or TMDB API.
- Users are responsible for any streaming accessed outside this platform.
